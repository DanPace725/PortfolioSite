ACE Framework App

The ACE (Action-Centered Engagement) Framework App is a task-tracking and rewards system built with Next.js and Supabase. It encourages users to take consistent actions toward their goals by providing immediate feedback, XP accumulation, skill progression, and rewards.

Technologies: Next.js, Supabase, Vercel, Tailwind CSS

  • 📊 Dashboard to view progress, actions, and rewards
  • ✔️ Log actions to earn XP and track goals
  • 🏆 Reward system for task completion
  • 🔐 Supabase authentication
  • 📱 Responsive design
